# Vision stratégique enchères inversées (validée client 2026-07-19)
Modèle 5 étages : abonnements récurrents (fournisseurs 0-249€, acheteurs 149-990€) + CPC prépayés (micropaiements INTERNES, jamais Stripe <25€) + frais organisateur par consultation (39-1490€) + services premium CPC (rapports 10, benchmark 10-20, API 20-50) + licences territoriales marque blanche (3-15k€ setup, 500-2500€/mois).
Règles : jamais pay-per-bid ni pay-to-win ; 4 types CPC (payés 24 mois / inclus 3-6 mois / bonus / solidaires, ordre de consommation expiration→bonus→inclus→payés) ; recharge auto opt-in plafonnée ; score de liquidité avant enchère (0-1 fournisseur=négociation, 2=scellée, 3+=enchère) ; consultations récurrentes + bibliothèque de modèles ; data agrégée anonymisée monétisable ; KPI marge contribution/consultation ; part récurrent >60%.
Phases : P1 rentabilité unitaire (fait en grande partie) → P2 automatisation (recharge auto, modèles, récurrence) → P3 multi-tenant marque blanche → P4 intelligence d'achat IA.
